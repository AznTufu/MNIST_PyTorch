📊 Informations sur le modèle

Ce modèle CNN a été entraîné sur le dataset MNIST avec les caractéristiques suivantes :

  • Architecture : CNN avec 3 blocs convolutionnels + couches fully connected
  • Augmentation de données : rotation, translation, perspective
  • Normalisation : [-1, 1] avec mean=0.5, std=0.5
  • Dropout et BatchNorm pour la régularisation
